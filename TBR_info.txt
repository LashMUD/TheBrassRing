The repository holds directories based on the Dead Souls mudlib. The directory
structure has been kept intact and contains files from the mudlib that are 
modified in creating The Brass Ring mud. See file headers and look in the 
/doc directory for notes and modifications. The Brass Ring specific docs 
will be labelled as TBR_x.txt

See http://www.dead-souls.net for more information.
